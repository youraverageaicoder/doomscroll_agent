# doomscroll_agent
A python script made by a child to encourages you to doomscroll more on instagram, Every 2 minutes it will send a message. It has 50 messages preinstalled and after 10 mins not doomscrolling, it wil open instagram reels for you.

# How to install:
Dependancy install: pip install pywin32 win10toast



Download the script and run it with python.



Works with windows 10 and 11
